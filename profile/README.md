# CommunityProjectTeam

커뮤니티 종합 관리 매니지먼트 웹 서비스 개발

## 프로젝트 소개

마스토돈 기반 커뮤니티를 위한 통합 관리 시스템입니다.

커뮤니티 운영진의 반복적인 수동 작업을 자동화하고, 유저에게 직관적인 활동 환경을 제공합니다.

### 주요 기능

- 마스토돈 OAuth 연동 로그인
- 유저 프로필 관리
- 휴식/경고/밴 관리 자동화
- 공지사항 및 이벤트 관리
- 출석 체크 자동화
- 활동량 모니터링

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | HTML5, CSS (TailwindCSS), Vanilla JS, Axios |
| Backend | Python, Flask |
| Database | PostgreSQL |
| Bot | Python, Mastodon.py |
